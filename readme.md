# Python Google Cloud Platform Demo
### run 
1)  Use python 3.6 +
2) Create virtual environment 
```sh
python -m venv env 
```
3) A) Activate Windows:
```sh
env\Scripts\activate 
```
 3) B) Linux/Mac 
 ```sh
 source env/bin/activate
```
4) Install libraries:
 ```sh
 pip install -r requirements.txt
```
5) run flask app
 ```sh
 python server.py
```

Repeat step 3 every time
